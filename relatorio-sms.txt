O trabalho Gerenciamento de transmissão de mensagens curtas foi desenvolvido em c++ e todas as entradas são feitas pelo teclado. A parte inicial do código consiste em criar as estruturas para armazenamento dos dados, definição das variáveis e definição de senha padrão (“123456”, é possível alterá-la apenas no código) para o acesso do administrador da base de telefones celulares. Após a execução do programa é exibido na tela o menu inicial, quando selecionado a primeira opção é exigida a senha, caso for digitado uma senha diferente da cadastrada o programa retorna ao menu inicial, caso contrário é permitido cadastrar um novo terminal celular, ver todos os terminais cadastrados ou voltar. Caso o usuário tente cadastrar qualquer dado fora dos padrões solicitados ele é encaminhado novamente ao menu do administrador.
	Feito os cadastros e retornando ao menu inicial, é permitido a transmissão de mensagens, acessando a segunda opção do menu e depois inserindo o celular (DDI + DDD + número) sem separação por espaços e a senha cadastrada nele. Após inserir algum número e senha cadastrados no banco de dados do software, é permitido ler mensagens recebidas ou enviadas e escrever uma nova mensagem, no início da tela é exibido a quantidade de mensagens recebidas e a de não lidas. Caso o usuário queira ler alguma mensagem, ele é levado a um outro menu com todas as mensagens enumeradas por ordem de chegada, basta informar o número da mensagem para ter acesso a todas as informações dela, como terminal do remetente horário do recebimento e conteúdo da mensagem. Caso o usuário queira enviar para um número não cadastrado ou a mensagem ultrapasse 160 caracteres, ele é retornado para o menu de transmissão de mensagens.
	Por fim voltando ao menu principal do programa é possível finalizá-lo apertando qualquer tecla. Esse programa é de fácil manuseio, contém as informações necessárias para cada passo do usuário.
	Foi utilizado a função fflush_stdin() manualmente para tornar o código mais limpo, pois em muitos momentos os dados se perdiam e viravam lixo. Também foi importada a biblioteca <time.h> para extração do tempo exato do sistema e vários comandos system(“cls”) para tornar o programa mais legível. O programa foi desenvolvido no Windows utilizando a IDE Devc++.
